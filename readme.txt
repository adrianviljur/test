Tools used in this solution:

- C9.io: It is the IDLE i have used to deploy the solution.
- NPM: To install different libraries (packages), like bower, express, helmet etc..
- Express: To make a server with Node.JS
- Helmet: To enhanced security in express server.
- Bower: A libraries manager i have used to install AngularJS, angular-route, bootstrap and Jquery.
- AngularJS: Get esiest the development of the visual interface, using models to change content in real time.
- Angular-route: To use views with angular.
- Bootstrap: To have enhanced visuals
- Jquery: Just because it is a bootstrap requeriment.
